Программная система NNSys 0.1 предназначена для моделирования процессов обучения и функционирования многослойных нейронных сетей прямого распространения сигнала с сигмоидальными функциями активации (многослойных персептронов).

Вычисления внутри слоёв распараллелены по нейронам с помощью технологии OpenMP. Реализовано пять алгоритмов обучения на базе градиентного метода (классический алгоритм обратного распространения ошибки и его модификации). Можно использовать три критерия останова процесса обучения: 1) достижение необходимого минимума среднеквадратичной ошибки обучения; 2) достижение необходимого минимума нормы градиента функции ошибки обучения; 3) возрастание ошибки обобщения. Последний критерий называется критерием раннего останова. Его применение позволяет предотвращать переобучение нейронной сети.

В дальнейшем планируется добавить генетический алгоритм обучения, алгоритм предобучения на базе глубоких сетей доверия (Deep Belief Networks), а также использовать технологию OpenCL вместо OpenMP для параллельных вычислений.

Ниже описаны основные возможности программной системы NNSys 0.1 (исчерпывающее описание возможностей текущей версии NNSys см. в документе users_guide_ru.pdf).

1. Создание многослойных персептронов с произвольным количеством слоёв и нейронов в слоях.

2. Использование двух видов функции активации для нейронов слоя — сигмоидальной и линейной.

3. Ускорение нейросетевых вычислений на многоядерном ЦПУ за счёт использования технологии OpenMP.

4. Обучение многослойных персептронов пятью возможными алгоритмами: стохастическим (онлайн) обратным распространением ошибки, Incremental Delta-Bar-Delta, пакетным обратным распространением ошибки, «упругим» обратным распространением ошибки, алгоритмом на базе метода сопряжённых градиентнов.

5. Возможность использовать комбинацию трёх критериев останова процесса обучения: 1) по достижению необходимого минимума среднеквадратичной ошибки обучения; 2) по достижению необходимого минимума нормы градиента функции ошибки; 3) при возрастании ошибки обобщения.

6. Вычисление как среднеквадратичной ошибки, так и ошибок классификации или регрессии при тестировании многослойного персептрона.

7. Возможность преобразовывать данные для обучения и/или тестирования из формата CSV во внутренний формат обучающего множества и обратно.

Программная система NNSys 0.1 поставляется по лицензии GPL версии 3. В поставку включёны файлы с исходным кодом программы, а также руководство пользователя на русском языке (перевод на английский планируется в скором будущем).

Для компиляции программы необходимы: 1) любой компилятор C++, поддерживающий стандарт ISO/IEC 14882:1998; 2) установленная библиотека Qt не ниже версии Qt 4.8.
